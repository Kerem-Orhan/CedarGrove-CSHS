/projects

stuff our members built.

A sample of projects from CSHS members this year — from hackathon hacks thrown together in 8 hours to semester-long passion projects. Many of these started from a "what if we…" at a Tuesday meeting.

//Class Favorite - Kerem Orhan

CopyPastePlus

A simple program that automatically types anything pasted in to simulate human typing. Also automatically makes typos and fixes them. Can adjust speed and typo rate. Built in less than 5 hours. Loved by students

React WebRTC Firebase
//Data Organization - Kerem Orhan

DataGrapher

A browser-based tool that transforms CSV files and spreadsheet data into interactive network graphs to visually represent patterns and data.

Python Z3 Flask
// GAME · GAMEJAM - Kevin Guan

Recursive Dungeon

A roguelike where every room contains a smaller version of the same dungeon. Procedurally generated, surprisingly playable. Final boss fight is in main().

Godot GDScript
// ML · RESEARCH - Julien Patel

CafeteriaCam

A computer-vision model that estimates lunch-line wait times from a live webcam feed. Helps students figure out when to leave class early-ish (for hall-pass approved reasons).

PyTorch YOLOv8 FastAPI
// HARDWARE · CLUB - Ethan Hartman

Door Sensor Network

ESP32-based sensors on classroom doors that report occupancy to a dashboard. Originally a prank, now actually used by faculty for fire-drill headcounts.

ESP32 MQTT Grafana
// WEB · TOOL - Peter Clarkson

LaTeX Live

A browser-based LaTeX editor with split-pane preview. Made for students writing physics and math papers without wanting to install a TeX distribution.

TypeScript WASM CodeMirror
// CYBERSEC · CTF - Rahul Solanki

PicoCTF Writeups

Collected solutions and walkthroughs from this year's PicoCTF competition. The team placed in the top 5% nationally.

Binary Analysis Cryptography
// MOBILE · COMMUNITY - Tim Antony

LostFound

A simple iOS/Android app for the school's lost-and-found. Take a photo, tag the location, ML matches it to "looking for" posts. Used mostly for AirPods.

React Native Supabase
// COMP. PROG.

USACO Training Group

Not a project but a programming team — members met weekly to grind USACO and Codeforces problems. Three members made USACO Gold this year.

C++ Algorithms
