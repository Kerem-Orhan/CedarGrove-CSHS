/resources

where to actually learn.

A curated list of resources our members have actually used and recommend. Not a comprehensive directory — just stuff that works.

Getting started

FOUNDATIONAL

CS50x

Harvard's intro to computer science. Free, rigorous, and the best starting point if you want a real foundation.

PRACTICE

Exercism

Free coding exercises in 60+ languages with mentor feedback. Best place to actually get your code reviewed.

WEB DEV

The Odin Project

Full-stack web development curriculum. Project-heavy, opinionated, and free. Goes from zero to deployable apps.

TOOLING

The Missing Semester

MIT lectures on the tools nobody teaches you — shell, Git, Vim, debugging. Worth a weekend.

Competitions & contests

ALGORITHMS

USACO

US Computing Olympiad. Four contests per year, Bronze through Platinum divisions. Standard practice for competitive programming.

SECURITY

picoCTF

Cybersecurity capture-the-flag for high schoolers. Annual competition plus a free year-round practice gym.

DEFENSE

CyberPatriot

National youth cyber defense competition. Teams secure virtual systems against simulated attacks. We send teams every fall.

CONTESTS

Codeforces

Weekly competitive programming contests with a global rating system. Where to go after you've mastered USACO Silver.

Deep dives

MACHINE LEARNING

fast.ai

Practical deep learning course. Code-first approach. You'll build real models in week one instead of doing matrix math for a month.

CURRICULUM

Teach Yourself CS

A complete self-taught CS degree, with the best resource picked for each subject. Aspirational and worth bookmarking.

SYSTEMS

Nand2Tetris

Build a computer from NAND gates up to a working OS and Tetris game. Removes all the magic. Free, follow-along course.

INTERVIEW PREP

Coding Interview University

A comprehensive prep list for software-engineering interviews. Useful if you're thinking about internships or college CS programs.

Have a resource we should add?

If something genuinely helped you learn, let us know and we'll add it. We only put up stuff that actual members have actually used.
